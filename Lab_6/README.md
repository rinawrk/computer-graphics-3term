## Лабораторная работа №6:
### Настройка освещения с использованием шейдерных программ

**Цель:** настроить освещение 3D-модели в OpenGL с помощью шейдеров и реализовать базовую модель освещения Фонга.

### Основные идеи
В этой лабораторной работе к уже импортированной 3D-модели добавляется освещение.  
Для этого в шейдерах реализуются основные компоненты модели Фонга: фоновое, рассеянное и зеркальное освещение.  
Дополнительно задаются свойства источника света и материала поверхности, а также используются нормали модели для корректного расчёта освещения.

### Что реализовано
- реализовано освещение по модели **Фонга**:
  - **ambient** - фоновая подсветка, чтобы объект не был полностью тёмным;
  - **diffuse** - светотень в зависимости от угла между нормалью и направлением к источнику света;
  - **specular** - блики, зависящие от положения камеры и параметра блеска (*shininess*);
- используются **нормали вершин** модели для корректного расчёта освещения;
- в вершинный шейдер добаваны:
  - мировая позиция фрагмента `FragPos`;
  - нормаль `Normal`;
  - матрица нормалей `normalMatrix` для корректного преобразования нормалей;
- во фрагментном шейдере добавлены структуры:
  - **Light** - параметры источника света;
  - **Material** - свойства материала поверхности;
- в основном коде задаются параметры:
  - источника света (позиция и интенсивности) `light.*`;
  - материала (цвет и блеск) `material.*`;
  - позиции камеры `viewPos` для расчёта зеркального освещения;
- в качестве 3D-объекта используется импортированная через **Assimp** модель робота **М20Ц**;
- сохранено управление камерой для осмотра сцены.

### Особенности реализации
Для корректного освещения при преобразованиях модели используется **матрица нормалей `normalMatrix`**.  
Она позволяет правильно преобразовывать нормали при поворотах и масштабировании объекта.  
Параметры света и материала задаются в основном коде через `uniform`-переменные и передаются в шейдерную программу перед отрисовкой модели.  

### Управление

- **W / A / S / D** - перемещение камеры;
- **мышь** - поворот камеры;
- **Esc** - выход.

### Результат
Реализовано освещение импортированной 3D-модели по модели **Фонга**.  
Модель корректно отображается в сцене с фоновым, рассеянным и зеркальным освещением, а управление камерой позволяет осматривать результат с разных ракурсов.
